# Hugo Januário | Portfolio Profissional

Este repositório contém o código-fonte do portfolio pessoal de Hugo Januário, Desenvolvedor Backend e Entusiasta de Infraestrutura. O projeto foi projetado com uma estética "Systems/Tech", focada em performance, código limpo e uma experiência de usuário premium.

## 🚀 Tecnologias Utilizadas

Para garantir o máximo controle sobre o design e performance, o projeto foi construído utilizando tecnologias web fundamentais:

- **HTML5**: Estrutura semântica para melhor SEO e acessibilidade.
- **Vanilla CSS**: Estilização personalizada sem frameworks, utilizando CSS Grid, Flexbox e Variáveis (Custom Properties).
- **Vanilla JavaScript**: Interatividade e animações personalizadas (efeito de digitação no terminal, menu mobile, scroll reveal).
- **Google Fonts**: Tipografia moderna utilizando as fontes *Outfit* e *JetBrains Mono*.
- **Font Awesome**: Ícones vetoriais para links sociais e elementos visuais.

## ✨ Funcionalidades

- **Terminal Interativo**: Uma simulação de terminal no Hero que digita comandos reais de backend.
- **Design Responsivo**: Totalmente otimizado para dispositivos móveis, tablets e desktops.
- **Glassmorphism**: Efeitos visuais modernos nas seções de projetos.
- **Scroll Reveal**: Animações suaves que revelam o conteúdo conforme o usuário navega.
- **Timeline de Carreira**: Organização clara da trajetória profissional.

## 🛠️ Como Executar o Projeto

Como o portfolio foi construído com tecnologias nativas, não é necessário rodar comandos de instalação complexos.

1. Clone o repositório ou baixe os arquivos.
2. Certifique-se de que a estrutura de pastas esteja correta (incluindo a pasta `Image`).
3. Abra o arquivo `index.html` em qualquer navegador moderno.

*Alternativamente, se desejar rodar em um servidor local para testar performance:*
```bash
# Caso tenha o Python instalado
python -m http.server 8000
```
